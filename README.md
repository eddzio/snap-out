# Snap Out

A Chrome extension that nudges you to stop scrolling LinkedIn and Facebook after a configurable time limit.

## Features

- Tracks active time on LinkedIn and Facebook (pauses when tab is hidden or idle)
- Shows a fullscreen overlay after your time limit is reached
- Configurable time limit (1 min, 3 min, or custom)
- Configurable popup message
- Quick links to redirect you to productive sites
- "5 more minutes" dismiss button that resets the timer
- Settings page to manage everything

## Install

1. Clone this repo
2. Open `chrome://extensions/` in Chrome
3. Enable **Developer mode** (top right)
4. Click **Load unpacked** and select this folder
5. Visit LinkedIn or Facebook and the timer starts

## Configure

Right-click the extension icon → **Options**, or go to `chrome://extensions/` → Snap Out → **Details** → **Extension options**.

From there you can:
- Switch between 1 min / 3 min / custom time limits
- Change the popup message
- Add or remove quick links
